# Πρόγραμμα Φορτώσεων v4

Απλή προσωπική εφαρμογή για GitHub Pages.

## Νέα στην έκδοση v4

- Πραγματικό autocomplete σε όλα τα κελιά (δουλεύει και σε κινητό).
- Αυτόματη μετατροπή όλων των ονομασιών σε ΚΕΦΑΛΑΙΑ.
- Autocorrect προτάσεις για κοντινές/λανθασμένες ονομασίες.
- Τικ ✓ δίπλα σε κάθε εβδομαδιαία εγγραφή για να σημειώνεται ότι έφυγε το βυτίο.
- Στο ΑΛΑΤΙ εμφανίζονται άμεσα οι επιλογές ΕΛΛΗΝΙΚΕΣ ΑΛΥΚΕΣ και ΔΑΚΑΡΙΔΗΣ.
- Σε κινητό εμφανίζεται μία ημέρα κάθε φορά, με επιλογέα ημέρας, ώστε να μη χρειάζεται zoom.
- Οι φωτογραφίες δεν αποθηκεύονται.

## Ανέβασμα στο GitHub Pages

1. Δημιούργησε νέο repository στο GitHub.
2. Ανέβασε όλα τα αρχεία του ZIP στη ρίζα του repository.
3. Άνοιξε Settings → Pages.
4. Επίλεξε Deploy from a branch, branch `main`, folder `/root`.
5. Περίμενε λίγα λεπτά και άνοιξε τη διεύθυνση που θα εμφανιστεί.

Τα δεδομένα αποθηκεύονται τοπικά στον browser της συσκευής. Χρησιμοποίησε την καρτέλα «Αντίγραφο ασφαλείας» για να κρατάς αντίγραφο.


## Αλλαγή v4
Στο κινητό εμφανίζονται ταυτόχρονα όσες περισσότερες ημέρες χωρούν στην οθόνη. Ο πίνακας μετακινείται οριζόντια χωρίς να χρειάζεται επιλογή μίας ημέρας ή συνεχές zoom.

## Έκδοση v5
- Διορθωμένο autocomplete από τα πρώτα γράμματα.
- Βελάκι ▾ δίπλα στα σχετικά κελιά για άμεση επιλογή από ολόκληρη τη λίστα.
- Στο εβδομαδιαίο οι προτάσεις προέρχονται από τη λίστα Πελατών, ενώ στο ΑΛΑΤΙ εμφανίζονται οι δύο προκαθορισμένες επιλογές.

## Έκδοση v6
- Δίπλα στην ένδειξη «ΕΒΔΟΜΑΔΑ» εμφανίζεται αυτόματα το σύνολο των συμπληρωμένων βυτίων.
- Οι λίστες πελατών, δεξαμενών, μεταφορέων και λοιπών ονομασιών αποθηκεύονται αυτόματα καθώς γράφεις, καθώς και όταν κλείνεις ή αλλάζεις καρτέλα.
- Νέα έκδοση cache ώστε οι αλλαγές να εμφανίζονται αμέσως μετά την ενημέρωση στο GitHub Pages.


## Έκδοση v7
- Η εισαγωγή φωτογραφίας περνά πλέον αποκλειστικά ονόματα πελατών.
- Η αναγνώριση συγκρίνει το κείμενο της φωτογραφίας με τη λίστα «Πελάτες / εταιρείες» και αγνοεί ημερομηνίες, προϊόντα, ποσότητες, τίτλους και λοιπό κείμενο.
- Γίνεται αυτόματη δοκιμή περιστροφής της φωτογραφίας όταν δεν εντοπίζονται αρκετοί πελάτες.
- Στα κελιά εισάγονται μόνο έγκυρα ονόματα που υπάρχουν ήδη στη λίστα πελατών.
