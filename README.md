# Πρόγραμμα Φορτώσεων

Μικρή προσωπική web εφαρμογή για εβδομαδιαίο και καθημερινό πρόγραμμα φορτώσεων.

## Τι κάνει

- Πίνακες που προσεγγίζουν τη δομή των φωτογραφιών Excel.
- Χειροκίνητη επεξεργασία όλων των κελιών.
- Autocomplete από λίστες πελατών, δεξαμενών και μεταφορέων.
- Ανάγνωση φωτογραφίας με Tesseract.js μέσα στον browser.
- Οι φωτογραφίες δεν ανεβαίνουν και δεν αποθηκεύονται.
- Το πρόγραμμα αποθηκεύεται τοπικά με `localStorage`.
- Εξαγωγή και επαναφορά αντιγράφου ασφαλείας JSON.

## Δημοσίευση στο GitHub Pages

1. Δημιούργησε νέο repository στο GitHub.
2. Ανέβασε όλα τα αρχεία αυτού του φακέλου στη ρίζα του repository.
3. Άνοιξε **Settings → Pages**.
4. Στο **Build and deployment**, επίλεξε **Deploy from a branch**.
5. Επίλεξε branch `main` και φάκελο `/ (root)`.
6. Πάτησε **Save**.

Μετά από λίγο η εφαρμογή θα είναι διαθέσιμη στη διεύθυνση που θα εμφανίσει το GitHub Pages.

## Σημαντικό για το OCR

Το OCR δημιουργεί προσχέδιο κειμένου. Επειδή οι φωτογραφίες είναι λοξές και περιέχουν πολλά μικρά κελιά, χρειάζεται πάντα έλεγχος. Για καλύτερο αποτέλεσμα:

- φωτογράφισε οριζόντια και ίσια,
- γέμισε το κάδρο μόνο με τον πίνακα,
- απόφυγε αντανακλάσεις,
- χρησιμοποίησε υψηλή ανάλυση.

## Ιδιωτικότητα

Η εικόνα περνά απευθείας στο Tesseract.js στη συσκευή. Δεν αποθηκεύεται από την εφαρμογή. Τα δεδομένα του προγράμματος παραμένουν στον συγκεκριμένο browser, εκτός αν εξαχθεί χειροκίνητα αντίγραφο ασφαλείας.

## Δομή εβδομαδιαίου προγράμματος

Κάθε ημέρα εμφανίζεται ως κάθετη στήλη. Μέσα σε κάθε στήλη υπάρχουν σταθερά:

- **ΥΠΟΧΛΩΡΙΩΔΕΣ ΝΑΤΡΙΟ:** 15 κενά κελιά πελατών
- **ΥΔΡΟΧΛΩΡΙΚΟ ΟΞΥ:** 2 κενά κελιά
- **ΑΛΜΗ:** 2 κενά κελιά
- **ΑΛΑΤΙ:** 2 κενά κελιά

Όλα τα κελιά συμπληρώνονται χειροκίνητα ή μέσω του OCR και αποθηκεύονται τοπικά στη συσκευή.
