# Πρόγραμμα Φορτώσεων v1.5.1 — κοινόχρηστη έκδοση

Η εφαρμογή αποθηκεύει το πρόγραμμα τοπικά και το συγχρονίζει στο Supabase. Όποιος ανοίγει το ίδιο GitHub Pages link βλέπει και επεξεργάζεται το ίδιο πρόγραμμα. Οι φωτογραφίες χρησιμοποιούνται μόνο προσωρινά για OCR και δεν αποθηκεύονται.

## Υποχρεωτική ρύθμιση Supabase

1. Άνοιξε το project `ccpthfogsrenwlulkabf` στο Supabase.
2. Πήγαινε **SQL Editor → New query**.
3. Άνοιξε το αρχείο `supabase-setup.sql`, αντέγραψε όλο το περιεχόμενο και πάτησε **Run**.
4. Ανέβασε όλα τα αρχεία αυτού του φακέλου στη ρίζα του GitHub repository.
5. Περίμενε 1–2 λεπτά και άνοιξε ξανά το GitHub Pages link.

Στην επάνω δεξιά ένδειξη πρέπει να εμφανίζεται **«Κοινόχρηστο • online»**. Αν εμφανίζεται «έλεγξε το Supabase setup», συνήθως δεν έχει εκτελεστεί το SQL ή το Project URL και το publishable key δεν ανήκουν στο ίδιο project.

## Σημείωση ασφάλειας

Η παρούσα έκδοση είναι σκόπιμα κοινόχρηστη: οποιοσδήποτε έχει το link μπορεί να δει και να αλλάξει τα δεδομένα. Το publishable key επιτρέπεται να βρίσκεται στον browser, ενώ το secret key δεν πρέπει ποτέ να μπει στον κώδικα.
