# Πρόγραμμα Φορτώσεων — πλήρης έκδοση v1.0

Αυτή είναι η ενιαία έκδοση για upload στο ίδιο GitHub repository. Περιέχει όλες τις αλλαγές που συμφωνήθηκαν έως τώρα.

## Περιλαμβάνει

- Εβδομαδιαίο πρόγραμμα 7 ημερών με κάθετες στήλες.
- 15 θέσεις ΥΠΟΧΛΩΡΙΩΔΟΥΣ ΝΑΤΡΙΟΥ, 2 ΥΔΡΟΧΛΩΡΙΚΟΥ ΟΞΕΟΣ, 2 ΑΛΜΗΣ και 3 ΑΛΑΤΙΟΥ ανά ημέρα.
- Τικ αναχώρησης δίπλα σε κάθε βυτίο.
- Αυτόματο σύνολο βυτίων εβδομάδας.
- Αλλαγή ημερομηνιών χωρίς διαγραφή του προγράμματος.
- Ξεχωριστό κουμπί καθαρισμού εβδομάδας.
- Καθημερινό με: πελάτη, δεξαμενή, μεταφορέα, ποσότητα, ημερομηνία/βάρδια, ώρα φόρτωσης, υπεύθυνο.
- Καρτέλα «Σειρά φορτώσεων» με δεξαμενή, αρχική/τελική στάθμη, ενεργά, σόδα και ολοκλήρωση.
- Autocomplete με πληκτρολόγηση ή βελάκι επιλογής σε εβδομαδιαίο και καθημερινό.
- Ονόματα πελατών σε κεφαλαία και πλήρης εμφάνιση μεγάλων ονομάτων στο κινητό.
- Ειδικές επιλογές ΑΛΑΤΙ: ΕΛΛΗΝΙΚΕΣ ΑΛΥΚΕΣ και ΔΑΚΑΡΙΔΗΣ.
- Αυτόματη αποθήκευση πελατών, δεξαμενών, μεταφορέων και υπευθύνων.
- OCR φωτογραφίας που κρατά μόνο πελάτες από την υπάρχουσα λίστα.
- Η φωτογραφία δεν αποθηκεύεται.
- Τοπική αποθήκευση και αντίγραφο ασφαλείας JSON.
- Καθαρισμός παλιάς cache/service worker, ώστε να μη φορτώνει προηγούμενη έκδοση.

## Ανέβασμα στο GitHub Pages

1. Διέγραψε ή αντικατάστησε όλα τα παλιά αρχεία του repository.
2. Ανέβασε όλα τα αρχεία αυτού του ZIP στη ρίζα του repository.
3. Περίμενε 1–2 λεπτά και άνοιξε ξανά τη σελίδα.
4. Στην κορυφή πρέπει να φαίνεται `v1.0`.

Τα παλιά δεδομένα των εκδόσεων `loadingPlanner.v1`, `v2` και `v3` μεταφέρονται αυτόματα στη νέα έκδοση.
