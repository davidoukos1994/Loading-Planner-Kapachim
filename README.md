# Πρόγραμμα Φορτώσεων v9

Προσωπική εφαρμογή για GitHub Pages.

## Νέα στη v9
- Η φωτογραφία του εβδομαδιαίου συμπληρώνει μόνο κελιά πελατών.
- Η φωτογραφία του καθημερινού συμπληρώνει αποκλειστικά τη στήλη **ΠΕΛΑΤΗΣ**.
- Δεν αλλάζουν δεξαμενή, μεταφορέας, ποσότητα, βάρδια, ώρα ή υπεύθυνος.
- Το autocomplete λειτουργεί σε πελάτη, δεξαμενή, μεταφορέα και υπεύθυνο στο καθημερινό και εμφανίζεται δίπλα στο ενεργό κελί.
- Όλες οι προηγούμενες λειτουργίες και τα αποθηκευμένα δεδομένα διατηρούνται.

Ανέβασε όλα τα αρχεία στον βασικό φάκελο του GitHub repository και ενεργοποίησε το GitHub Pages.
